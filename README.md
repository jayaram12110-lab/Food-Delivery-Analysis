# 🍔 Food Delivery Analysis

## 📌 Project Overview
This project analyzes a food delivery dataset using Python to understand customer ordering behavior, restaurant preferences, ratings, and order trends.

## 🎯 Objectives
- Analyze customer ordering patterns
- Compare average order values across cities
- Study restaurant types and cuisines
- Analyze customer ratings
- Explore payment methods and discounts

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- VS Code

## 📂 Dataset
- 50,000 food delivery records
- Cleaned using Pandas
- Includes customer, restaurant, order, and rating information

## 📊 Analysis Performed
- Data Cleaning
- Missing Value Check
- Duplicate Value Check
- Descriptive Statistics
- Average Order Value by City
- Ratings Distribution
- Restaurant Type Analysis
- Cuisine Analysis
- Payment Method Analysis

## 📈 Key Insights
- Customer ordering behavior differs across cities.
- Premium restaurants generally have higher order values.
- Ratings are distributed between 1 and 5.
- Popular cuisines receive a large share of total orders.
- Discounts influence repeat customer behavior.

## 🚀 Conclusion
This project demonstrates how Python can be used for Exploratory Data Analysis (EDA) to generate business insights that help improve customer satisfaction and support better decision-making for food delivery platforms.
